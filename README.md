# stock-quotes
因为我喜欢用linux系统，通达信不能跨平台，还看不了全球的指数走势，所以就自己写了一个。

类似通达信一类软件,python+pyside6开发，可以实时看盘，查看全球主要指数，美股和港股的实盘。

A股板块、指数、个股k线均线资料均可查看，搜索。F10资料，主要指标，经营分析，3大报表，板块资金流。

还可以实时语音播报当前各种新闻
